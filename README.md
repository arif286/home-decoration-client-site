### About projects and features:

- A customer can choose their best service and can discuss about the service their needs.
- Customers can place order and see their service progress and get up to date.
- A Customer panel created for only customer get access and see their activities.
- Also an admin panel reserve for admin activities like add services, modify services, manage customer orders and monitor website activities.
- In this projects Rest api use for maintaining user interface.
- saving user information use mongo db database.
- It is a MERN stack and fully responsive web site.
